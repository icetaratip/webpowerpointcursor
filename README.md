# Cursor AI Presentation

เว็บนำเสนอเคสศึกษา **Anysphere & Cursor** (Vite + React)

## รันบนเครื่อง

ต้องมี [Node.js](https://nodejs.org/)

```bat
start.cmd
```

หรือ:

```bat
npm install
npm run dev
```

เปิด http://localhost:5173/

## Build

```bat
npm run build
npm run preview
```

## Deploy

- GitHub: https://github.com/icetaratip/webpowerpointcursor
- Vercel: เชื่อม repo นี้ → Framework **Vite** → Output `dist`
